# vfox-promtail

[![CI](https://github.com/jbox-web/vfox-promtail/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-promtail/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [promtail](https://github.com/grafana/loki)

## Install

```
mise plugin install promtail https://github.com/jbox-web/vfox-promtail.git
```
